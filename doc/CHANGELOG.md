# 更新日志
## v26.03b3
新增：
* 活动商店购买。可指定需要购买的物品与优先级，将会每天自动购买
* 单人演出与自动演出现已支持指定歌曲
* 自动演出现已支持随机选曲
* 每日单人演出现在支持可选追加一次 FC 演出与一次随机歌曲演出，分别用于完成「在演出中以 FULL COMBO 完成歌曲达到 10 次」和「在演出中完成 10 首歌曲」活动任务
* 优化跨平台兼容性（macOS、Linux）
* 优化返回首页行为
* 连接方式新增 USB，支持连接物理设备

修复：
* 修复自动编队在非活动期间卡住的问题

## v26.03b2
新增：
* 单人演出支持自动编队
* 单人演出与自动演出支持设置 AP 倍率
* 现在当使用脚本自动进行自动演出时，AP 倍率会被锁定在 0
* 新增支持自动领取所有任务奖励

修复：
* 修复了「修复了台服游戏更新导致的一直卡在「返回游戏首页」上」导致的一直卡在「返回游戏首页」上

已知问题：
* 非活动期间，自动编队会卡住

## v26.03b1
新增：
* MuMu 模拟器现已支持选择多开实例，默认使用第一个实例
* CLI 与 GUI 入口现已分离。iaa.exe 为 GUI 入口，iaa-cli.exe 为 CLI 入口
* CLI 现已支持 `run`（运行当前配置）与 `invoke`（显式运行一个或多个任务）两种启动方式，并补充了任务列表与配置列表命令

修复：
* 修复了自动演出会卡在 RANK UP 画面上
* 修复了脚本演出不支持 1280x720 以上等比例分辨率的问题
* 修复了「活动剧情」会卡在第一次进入活动页面后自动进入的剧情里
* 修复了有时候活动期间，启动游戏会一直卡在「返回游戏首页」上
* 修复了台服游戏更新导致的一直卡在「返回游戏首页」上
